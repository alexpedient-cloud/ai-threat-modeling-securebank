\# AI Threat Modeling – SecureBank LLM System



This project demonstrates threat modeling for a production LLM-powered loan approval system.



\## Methodology



The analysis applies:



\- STRIDE threat modeling

\- MITRE ATLAS attack mapping

\- AI-specific vulnerability analysis



\## Key AI Security Risks



\- Prompt injection attacks

\- Sensitive data leakage

\- Model inversion attacks

\- Resource exhaustion (AI DoS)

\- Unauthorized API access



\## Security Controls



\- Prompt sandboxing

\- Input validation

\- Output filtering

\- Rate limiting

\- Secure ML pipeline protections
